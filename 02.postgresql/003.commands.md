分类命令


让用户可以创建 schema:

    GRANT CREATE ON DATABASE db TO user;

创建 schema

    CREATE SCHEMA evtity

显示当前 schema

    SELECT current_schema();




常用命令

| 命令              | 解释 |
|-------------------|------------|
| \l                | list 显示数据库 |
| \c                | connect 切换数据库 |
| \dt               | 显示 public 的全部表 |
| \dt \*.\*         | 显示全部表 |
| \dt public.\*     | 显示 public 的全部表 |
| \dn               | 显示全部 Schema |



## 命令详解

### 显示全部 Schema 的表：

    \dt *.*

### 显示单个 Schema 的表：

    \dt evtity.*

### \dn 显示全部命名空间


    > \dn
    +--------+----------+
    | Name   | Owner    |
    |--------+----------|
    | evtity | gzh      |
    | msg    | gzh      |
    | public | postgres |
    +--------+----------+
