# Ruby Gems

RubyGems 是 Ruby 的包管理器，它提供一个分发 Ruby 程序和库的标准格式，还提供一个管理程序包安装的工具。

安装好 Ruby 之后，同时也就安装了 gem，你可以用下面的命令查看是否已经安装了 gem：

    $ gem -v
	3.1.2

一个 gem 就是一个模块（包）。

## gem 常用命令

### gem install

安装模块

    gem install GEMNAME

### gem uninstall

卸载模块

    gem uninstall GEMNAME

### gem list

列出本地已安装的模块：

    gem list --local

列出远程可用的模块：

    gem list --remote
