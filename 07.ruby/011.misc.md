## 遍历 Hash

    frequencies.each { |key, value| puts "#{key} #{value}" }

或者：

    frequencies.each_with_index do | element, index |
    	puts "#{element} is number #{index} in the array"
    end



## 遍历数组

如果要获取索引值，可以使用 `each_with_index`

    array.each_with_index do | element, index |
    	puts "#{element} is number #{index} in the array"
    end
