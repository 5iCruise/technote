# Bulma

## 响应式

### 断点（Breakpoints）

| 断点        | 尺寸（px） |
| ----------- | -------------- |
| mobile      |  ~ 768 |
| tablet      |  769 ~ 1023 |
| Desktop     |  1024 ~ 1215 |
| Widescreen  |  1216 ~ 1407 |
| FullHD      |  1408 ~ |



## Column

左右布局（左边栏，右边是主内容）基本结构：

```html
<section class="section">
  <div class="columns">
    <div class="column is-4-tablet is-3-desktop is-2-widescreen">
    </div>

    <div class="column">
    </div>
  </div>
</section>
```
