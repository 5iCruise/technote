# 3. 导航和菜单

本章介绍如何使用导航和菜单组件。


导航栏：

```
<nav class="navbar has-shadow">
  <div class="navbar-brand">
    <!-- Logo, tagline, and navbar-burger -->
  </div>
  <div class="navbar-menu">
    <!-- User name, dropdown menu -->
  </div>
</nav>
```




### 导航栏菜单

navigation's menu

导航栏菜单分两部分：

- navbar-start
- navbar-end
