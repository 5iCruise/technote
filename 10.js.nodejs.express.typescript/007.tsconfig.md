# tsconfig.json

## include

include 用一个数组，指定要编译的 TS 文件。可以使用匹配模式：

    * 匹配 0 个或多个字符
    ？匹配 1 个或多个字符
    **/ 递归匹配子路径

所以，通常你这样设定就可以了：

    "include": [
      "src/**/*"
    ]

include 可以和 files 参数一道使用。
