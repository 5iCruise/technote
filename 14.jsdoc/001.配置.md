# JSDoc 配置

使用参数 -c 指定配置文件，配置文件是 json 格式：

    jsdoc -c conf.json

### 实例1

jsdoc 默认的配置如下：

```js
{
    "plugins": [],
    "recurseDepth": 10,
    "source": {
        "includePattern": ".+\\.js(doc|x)?$",
        "excludePattern": "(^|\\/|\\\\)_"
    },
    "sourceType": "module",
    "tags": {
        "allowUnknownTags": true,
        "dictionaries": ["jsdoc","closure"]
    },
    "templates": {
        "cleverLinks": false,
        "monospaceLinks": false
    }
}
```
