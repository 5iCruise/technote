## Q: 怎样去掉自动生成 README.md？

Oclif 在发布的时候，会自动生成 README.md。这个功能在 package.json 中配置：

    "scripts": {
        "prepack": "yarn build && oclif manifest && oclif readme"
    }

因此，去掉 oclif readme，就能阻止自动生成 README.md 文件。
