# Certbot

免费的SSL证书

使用 snapd 安装 certbot：

    sudo apt install snapd
    sudo snap install core; sudo snap refresh core
    sudo snap install --classic certbot
    sudo ln -s /snap/bin/certbot /usr/bin/certbot

创建证书：

    sudo certbot --nginx
