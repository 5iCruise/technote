# 终端录制软件



## terminalizer

这是 JavaScript 开发的，为什么下载量好少，一周才几百个？

在几个环境中安装，只有 Mac 安装成功了。

使用方式：

    terminalizer record demo        # 生成demo.yml文件
    terminalizer render demo        # 生成gif文件
